# Smart Health & Diet Recommendation System

## Project Overview
This project is a full-stack web application built with Django that helps users track their health, get diet recommendations, plan meals, and interact with AI-based features like diet prediction and food image recognition.

## Features
- User registration and profile management
- Meal planner and diet predictor
- Health report and weekly health plan
- Chatbot for FAQs
- Food image upload and classification
- Quizzes for health awareness
- ML models for diet recommendation and food classification

## Tech Stack
- Backend: Django
- Frontend: HTML, CSS, Bootstrap, JavaScript
- Database: SQLite / MySQL
- Machine Learning: scikit-learn, TensorFlow/Keras
- Visualization: Chart.js

## How to Run
1. Clone the repository.
2. Create a virtual environment and install dependencies: `pip install -r requirements.txt`.
3. Run migrations: `python manage.py migrate`.
4. Create a superuser: `python manage.py createsuperuser`.
5. Run the server: `python manage.py runserver`.
